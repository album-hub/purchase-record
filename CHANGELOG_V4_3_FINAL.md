# Fansign Manager V4.3 Final

## 新增
- Quick Add 2.0
  - 欄位式快速新增，支援無限新增與刪除列
  - 保留「多人同通路」與「同一人多通路」兩種模式
  - 購買人與通路訂單皆可搜尋、下拉選擇
  - 每列可填數量、金額與備註
  - 快速新增的訂單預設為已付款
  - 保留文字貼上模式，支援 `$金額` 格式
  - 支援 Tab 與 Enter 連續輸入；最後一格按 Enter 會自動新增下一列
  - 建立前檢查重複訂單與通路剩餘數量

- 購買人訂單 Tree View
  - 購買人 → 藝人 → 通路 → 通路詳細內容
  - 支援展開 / 收合
  - 每層顯示訂單數量
  - 保留原本篩選：進行中、未付款、已付款、已封存、全部
  - 搜尋仍支援購買人、藝人、通路、批次與備註

## V4.3 完整功能
- Backup Center
- Quick Add 快速新增
- 配送 Checklist
- 通路別稱
- 購買人訂單 Tree View

## 建議測試
1. 購買人訂單頁是否以 Tree View 顯示。
2. 搜尋購買人名稱是否能展開出該購買人的藝人 / 通路 / 詳細內容。
3. 篩選「未付款 / 已付款 / 已封存」是否正常。
4. 配送 Checklist 是否仍可正常勾選。
5. Quick Add 建立後是否能在 Tree View 中看到新訂單。
6. Quick Add 兩種模式是否都能新增／刪除列並寫入金額、備註與已付款狀態。
7. 文字貼上模式的 `$金額` 是否正確寫入。
