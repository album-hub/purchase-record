# Fansign Manager V4.3.1

## 新增
- 配送 Checklist
  - 建立賣場
  - 下單
  - 出貨
  - 三個步驟可獨立勾選
  - 已付款 + 三個配送步驟完成後，會進入已封存

- 通路別稱
  - 在設定 > 通路管理可設定單一別稱
  - 通路顯示會出現「通路名稱（別稱）」
  - 快速新增同一人多通路時，可用別稱搜尋

## 建議測試
1. 設定 > 通路管理，替通路設定別稱，例如 MS。
2. 新增或搜尋通路訂單，確認顯示「通路名稱（MS）」。
3. 購買人訂單卡片勾選：建立賣場、下單、出貨。
4. 確認已付款 + 三項配送完成後，該筆訂單進入已封存。
5. 快速新增「同一人多通路」時，用別稱測試是否能找到通路。
