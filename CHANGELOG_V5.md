# Fansign Manager V5.0

## V5.3
- 購買人訂單保留原本 Tree View，另增「單人多通路訂單」匯集區。
- 可選取同一購買人的指定訂單建立匯集訂單，並設定、複製專屬賣貨便連結。
- 訂單確認中心拆分為待確認、收款資料尋找與確認封存三區。
- 可依姓名、付款時間、數量與金額尋找可能訂單，並以時間優先排序。
- 付款時間、數量與金額三項完成後，自動進入可搜尋的確認封存區。
- 快速新增購買人下拉顯示完整名單並保留搜尋。
- 快速新增支援「建立後繼續下一批」，可連續處理下一批資料。
- 同一次快速新增可建立多個不同通路區塊，各區塊保留完整欄位並以分隔線區分。
- 快速新增的數量／金額規則改為每個通路區塊獨立設定，不再共用同一份價格。
- 首頁收款與人數統計改為可收合區塊。
- 單人多通路訂單移至購買人訂單頁面最上方。

## 下一次更新規劃：小卡圖鑑
- 在底部主要功能區新增「小卡圖鑑」專區。
- 圖鑑項目可自行上傳小卡照片。
- 自動讀取通路訂單與完成專區的資料。
- 圖鑑只保留以下訂單資訊：藝人名稱、通路名稱、批次、專輯類型。
- 圖鑑項目進入完成專區後，自動補上所屬完成群組名稱。
- 照片需與原通路訂單保持關聯，避免訂單名稱或群組變更後資料不同步。

## V5.1
- 設定頁新增訂單確認中心，可確認付款時間、數量與金額。
- 通路訂單新增以今年為年度的月／日快速日期輸入。
- 購買日期可在同一欄輸入兩筆月日，例如 `0812 0815`，分別帶入開始與結束日期。
- 通路訂單卡片新增購買日期與簽售日期快速輸入按鈕。
- 「多人同通路」快速新增可同時輸入購買日期與簽售日期，並同步顯示於購買人訂單。

## V5.2
- 新增批次管理；通路訂單的批次改為可維護的下拉選單。
- 舊通路訂單中的批次會自動加入批次管理清單。
- 偵測到尚未分類的完成通路訂單時，自動提醒並可選擇既有群組或建立新群組。
- 通路訂單新增依數量批量修改購買人訂單金額，例如 `1=200 2=350`。
- 設定頁新增可展開的完整功能使用說明書。
- 新增完成專區，可將符合完成條件的通路訂單組成自訂名稱 Tree View。

## 新增與調整
- 快速新增支援統一數量／金額價目表。
- 快速新增數量提供加減按鈕，通路改為只顯示未登記完成訂單的下拉選單。
- 通路訂單改為「藝人 → 通路 → 批次 → 訂單」Tree View。
- 購買人訂單 Tree View 預設全部收合。
- 離開購買人訂單頁面時，自動清除展開狀態並全部收合。
- 通路與購買人依英文／注音順序排列。
- 首頁新增收款、人數及每位購買人筆數統計。
- 設定頁改為可展開／收合的 Tree View。
- 通路訂單可儲存賣貨便賣場連結。
- 通路與購買人訂單可一鍵複製賣貨便連結。
- 尚未設定賣貨便連結時，可從訂單卡片直接開啟輸入對話框。
- 未選取姓名建議時，自動帶入第一位符合者並顯示警示。
- 停用雙擊放大。
