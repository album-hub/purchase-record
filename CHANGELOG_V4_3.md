# Fansign Manager V4.3

## 新增
- Backup Center 備份中心
  - 手動建立備份
  - 自動備份保留最近 20 份
  - 備份列表顯示資料統計
  - 一鍵還原
  - 複製備份 JSON
  - 清空、匯入前自動建立安全備份

- 快速新增購買人訂單
  - 模式 A：多人同通路
  - 模式 B：同一人多通路
  - 支援數量解析：小美 2、小美、 小美 2張、小美 x2、小美+2
  - 支援 # 備註
  - 建立前預覽
  - 建立前自動備份

## 建議測試
1. 設定頁建立備份，確認備份中心出現紀錄。
2. 複製備份 JSON，確認下方文字框可接收。
3. 快速新增多人同通路，確認多筆訂單建立成功。
4. 快速新增同一人多通路，確認可用通路關鍵字建立多筆訂單。
5. 測試 # 備註是否正確存進訂單備註。
